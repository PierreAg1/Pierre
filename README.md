# Projet de Test GitHub

Ce dépôt est un projet simple pour pratiquer la gestion de dépôt sur GitHub. Il inclut un script Python minimal `main.py` qui réalise une addition entre deux nombres fournis par l'utilisateur.

## Contenu

- `main.py`: Script principal.
- `README.md`: Documentation du projet.
- `.gitignore`: Fichier pour exclure certains fichiers du dépôt.
- `requirements.txt`: Liste des dépendances (aucune ici, mais fichier inclus pour la pratique).
